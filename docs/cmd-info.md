The `sysl info` command provides information about the sysl binary.

## Usage

```bash
usage: sysl info
```

## Output

Example output:

```
Build:
  Version      : 1b2c16f0-feature/sysl-ui-2020-03-13T00:35:40Z
  Git Commit   : 1b2c16f0
  Date         : 2020-03-13T00:35:41Z
  Go Version   : unspecified
  OS           : unspecified
Runtime:
  GOOS/GOARCH  : darwin/amd64
  Go Version   : go1.13.7
```
