## Summary

The `sysl info` command provides information about the sysl binary.

## Usage

```
sysl info
```

## Output

Example output:

```
Build:
  Version      : v0.144.0
  Git Commit   : 4493a3e49ee3d95a6d4a78b16c9ccf712e2fb94d
  Date         : 2020-07-17T03:33:55Z
  Go Version   : go1.13.14 linux/amd64
  OS           : darwin/amd64
```
